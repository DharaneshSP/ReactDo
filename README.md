<div align="center">
A simplest React Based Task Manager
</div>

## Installation

- clone repository:
  
    `git clone https://github.com/DharaneshSP/ReactDo.git`
- Navigate to ReactDo
  
    `cd ReactDo`
- Install Dependencies
  
    `npm i`
- Start Application
  
    `npm start`
